III. Google Colab Work Sample: World Happiness Report Analysis using Python

This work sample demonstrates the use of Google Colab for conducting an exploratory data analysis on the World Happiness Report, a global survey correlating personal happiness with various quality of life factors. The dataset, spanning from January 1, 2015, to January 4, 2022, was sourced from Kaggle. The project highlights Python programming skills, particularly in data manipulation, analysis, and visualization. It includes various charts and graphs to represent the insights derived from the dataset effectively. This sample showcases the ability to leverage Google Colab's collaborative and interactive environment for advanced data analysis tasks.
